## What This Project Does

`cargo-workspace-unused-pub` is a Cargo subcommand that detects unused `pub` items across an entire Rust workspace. It fills a gap left by the built-in `dead_code` lint, which operates crate-by-crate and ignores `pub` items. It works by analyzing a SCIP index (generated by rust-analyzer) to find `pub` declarations with no references.

## Build & Test Commands

```sh
cargo build                # build
cargo test                 # run all tests
cargo test <test_name>     # run a single test
cargo run -- workspace-unused-pub [OPTIONS] [WORKSPACE]  # run as cargo subcommand
```

The binary is invoked as `cargo workspace-unused-pub`. It requires `rust-analyzer` on PATH to generate SCIP indexes.

## Architecture

The entire tool is a single file: `src/main.rs`. Detection runs in two passes:

1. **Pass 1** — Parse the SCIP index, collect all declarations (methods, functions, structs, enums, traits, constants, macros, etc.), then remove any symbol that has a non-definition occurrence (i.e., it's referenced somewhere). With `--skip-test-usages`, only non-test references count.

2. **Pass 2** — Filter remaining candidates to remove false positives: `fn main`/`#[test]`/`#[tokio::main]` functions, trait methods (detected via SCIP symbol path containment), items inside `#[cfg(test)]` modules (when `--skip-test-usages`), and non-pub struct fields.

Key helpers:
- `has_test_or_main_attr` — scans lines above a definition for test/main attributes
- `is_inside_cfg_test` — brace-counting to detect `#[cfg(test)]` scope
- `is_test_file` — checks if a path contains a `tests/` component

## Notable Design Choices

- The README mentions a Pass 3 (text-based grep fallback), but it has been removed from the code. The `--extensions` flag documented in the README is also no longer present.
- SCIP symbol strings encode the full path including trait names, so trait method detection works by checking if any trait symbol's descriptor appears as a substring of the method's symbol.
- The tool exits with code 2 if any unused pub items are found (useful for CI).
